# BDA CaseStudy

This repository contains a data analysis mini project on COVID-19 vaccine distribution across states.

## Files

- `covid_vaccine_statewise.csv` - source dataset for vaccine statistics by state
- `DSBDA Mini Project.ipynb` - Jupyter notebook containing data analysis, visualizations, and insights

## Notes

The notebook explores vaccine coverage, trends, and state-level comparisons using the dataset provided.
