# 💉 COVID-19 Vaccination Analysis in India
### 📊 Big Data Analytics

---

## 📌 Project Description
This project analyzes COVID-19 vaccination data across different states of India using data analytics techniques. The goal is to extract insights, identify trends, and visualize vaccination progress using Python-based tools.

The project demonstrates how big data analytics can help in understanding large-scale healthcare data and support informed decision-making.

---

## 🎯 Objectives
- Analyze state-wise vaccination data  
- Compare first and second dose administration  
- Identify patterns and trends  
- Visualize vaccination progress  
- Extract meaningful insights  

---

## 📂 Dataset Information
- **Dataset:** `covid_vaccine_statewise.csv`  
- **Type:** Structured (CSV)  
- **Description:** Contains state-wise vaccination data including:
  - Total doses administered  
  - First dose  
  - Second dose  

---

## 🛠️ Technologies Used
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Jupyter Notebook  

---

## ⚙️ Project Workflow
1. Data Collection  
2. Data Cleaning & Preprocessing

---

## 📈 Key Insights
- Identified states with highest vaccination rates
